## 포인터란?
- 메모리 주소를 저장하는 변수
- "값이 있는 위치"를 가리킴

---

## 포인터 선언
```c
int *p;
```

---

## 포인터 초기화
```c
int x = 10;
int *p = &x;
```

---

## 주소 연산자 `&`
```c
int x = 10;
int *p = &x; // x의 주소
```

---

## 역참조 연산자 `*`
```c
int x = 10;
int *p = &x;
printf("%d\n", *p); // 10
```

---

## 포인터로 값 변경
```c
int x = 10;
int *p = &x;
*p = 20; // x가 20으로 변경
```

---

## NULL 포인터
- 유효한 주소가 없는 포인터

```c
int *p = NULL;
```

---

## NULL 체크
```c
if (p != NULL) {
    printf("%d\n", *p);
}
```

---

## 포인터와 배열
- 배열 이름은 첫 요소의 주소

```c
int arr[3] = {1, 2, 3};
int *p = arr;
```

---

## 배열 이름이 포인터가 아닌 이유
- 대부분 표현식에서 배열 이름은 포인터처럼 변환(decay)
- 하지만 배열 자체 타입은 포인터 타입과 다름

```c
int arr[3] = {1, 2, 3};
printf("%zu\n", sizeof(arr));    // 12 (예: int 4바이트)
printf("%zu\n", sizeof(&arr[0])); // 8  (예: 64비트 포인터)
```

---

## 포인터 산술
```c
int arr[3] = {1, 2, 3};
int *p = arr;
printf("%d\n", *(p + 1)); // 2
```

---

## 포인터와 함수
- 함수에서 원본 변경 가능

```c
void inc(int *p) {
    (*p)++;
}
```

- 해제되지 않은 메모리 접근
- NULL 역참조
