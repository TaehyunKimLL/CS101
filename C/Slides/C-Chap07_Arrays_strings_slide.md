## 배열이란?
- 같은 타입의 값을 연속으로 저장
- 인덱스로 접근 (0부터 시작)

---

## 배열 선언
```c
int nums[5];
```

---

## 배열 초기화
```c
int nums[5] = {1, 2, 3, 4, 5};
```

---

## 인덱스 접근
```c
printf("%d\n", nums[0]); // 첫 번째 값
```

---

## 배열과 반복문
```c
for (int i = 0; i < 5; i++) {
    printf("%d ", nums[i]);
}
```

---

## 배열 크기 계산
```c
int n = sizeof(nums) / sizeof(nums[0]);
```

---

## 범위 주의
- 유효 인덱스: 0 ~ (크기-1)
- 범위를 넘으면 오류/UB

---

## 문자 배열과 문자열
- 문자열은 `char` 배열
- 끝에 `\0`가 필요

```c
char name[6] = "Alice"; // A l i c e \0
```

---

## 문자열 입력
```c
char name[20];
scanf("%19s", name);
```

---

## 문자열 출력
```c
printf("%s\n", name);
```

---

## 문자열 길이
```c
#include <string.h>
size_t len = strlen(name);
```

---

## 문자열 복사
```c
char a[20] = "hi";
char b[20];
strcpy(b, a);
```

---

## 문자열 비교
```c
if (strcmp(a, b) == 0) {
    printf("same\n");
}
```

---

## 안전한 문자열 처리
- 길이 제한을 항상 고려
- 버퍼 크기보다 큰 입력 금지
