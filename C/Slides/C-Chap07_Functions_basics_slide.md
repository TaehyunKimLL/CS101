## 함수란?

- 프로그램을 작은 단위로 분해

## 함수가 필요한 이유
- 코드 중복 제거
- 가독성/유지보수 개선
- 테스트 단위 분리

---

## 함수 선언 vs 정의
- 선언: 형태만 알림 (prototype)
- 정의: 실제 구현

```c
int add(int a, int b);   // 선언
int add(int a, int b) {  // 정의
    return a + b;
}
```

---

## 함수 호출
```c
int sum = add(3, 5);
```

---

## 매개변수와 인자
- 매개변수: 함수가 받는 변수
- 인자: 호출 시 전달하는 값

---

## 반환값
- `return`으로 값 반환
- 반환 타입과 일치해야 함

```c
double avg(int total, int count) {
    return (double)total / count;
}
```

---

## void 함수
- 반환값이 없는 함수

```c
void print_line(void) {
    printf("-----\n");
}
```

---

## 매개변수 전달 방식
- C는 **값에 의한 전달**
- 원본 값은 바뀌지 않음

```c
void inc(int x) { x++; }
```

---

## 값 전달 예시
```c
int a = 10;
inc(a);
// a는 여전히 10
```

---

## 포인터로 변경하기 (예고)
- 원본 변경은 포인터 필요
- 다음 챕터에서 자세히 다룸

---

## 중요한 정리: 포인터도 값 전달
- C에서 함수 인자는 항상 값 복사
- 포인터 인자도 주소값이 복사되어 전달됨
- 같은 주소를 가리키므로 원본 변경이 가능해 보임

```c
void set_zero(int *p) { *p = 0; }  // p 자체는 복사본
```

---

## 변수 범위 (scope)
- 지역 변수: 함수 내부에서만 유효
- 전역 변수: 파일 전체에서 유효

---

## 수명 (lifetime)
- 지역 변수: 함수 종료 시 소멸
- 전역 변수: 프로그램 종료 시 소멸

---

## 함수 분해 예시
```c
int read_int(void) {
    int x;
    scanf("%d", &x);
    return x;
}

int max(int a, int b) {
    return (a > b) ? a : b;
}
```
