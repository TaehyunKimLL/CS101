# 학습자용 실습 워크시트
## Windows 개발환경 & Command Line 기초 (통합본)

과목명: 프로그래밍 입문  
대상: 대학 신입생 / 프로그래밍 초심자  
실습 시간: 약 50~60분  
환경: Windows 10/11 + Windows Terminal + VS Code

---

## 실습 목표

이 워크시트를 완료하면 다음을 할 수 있습니다.

- Windows Terminal에서 Command Prompt와 PowerShell을 실행할 수 있다
- Command Line의 기본 개념을 이해한다
- 현재 위치를 확인하고 폴더를 이동할 수 있다
- 폴더와 파일을 직접 생성할 수 있다
- Tab 키로 자동 완성을 사용할 수 있다
- 터미널에서 Copy & Paste를 할 수 있다
- VS Code와 터미널을 함께 사용할 수 있다
- 에러 메시지를 보고 당황하지 않는다

---

- Windows Terminal 설치 확인
- Visual Studio Code 설치 확인
- 키보드로 직접 입력할 준비

⚠️ 복사 & 붙여넣기보다는 **직접 타이핑을 권장**합니다.

## 실습 2. 다른 쉘(Command Prompt / PowerShell) 실행하기

### 개념
Windows Terminal에서는 여러 종류의 쉘을 실행할 수 있습니다.

- Command Prompt: 기본적인 Windows 명령어
- PowerShell: 최신 Windows 표준 쉘

### 실습
1. 상단의 **▼ 버튼** 클릭
2. **Command Prompt** 선택 → 새 탭 확인
3. 다시 ▼ 클릭 → **PowerShell** 선택
